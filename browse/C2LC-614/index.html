    • Simon Bates commented 2022-03-09T11:33:00.826-0500

      We discussed keyboard shortcuts at the meeting with Bridges today.

      • We will add 3 new keyboard shortcuts
        • Select the loop action in the action panel
        • Move focus to the next loop block
        • Move focus to the previous loop block
      • For the number of iterations we will not provide specific keyboard shortcuts
        • For Switch ATs, such as The Grid 3, we can make a number pad style keyboard for editing the number of iterations
