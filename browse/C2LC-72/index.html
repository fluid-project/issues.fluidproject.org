    Description

    A digital canvas on which users can paint or draw - Generates sounds as strokes are being painted - Lets users share their canvas with others using a URL - Others can load a shared canvas and continue painting


    Details from co-design:

    A system for making shared visual (“splatter paintings”) and auditory/music art. The system would enable students to make shapes, optionally paired with sounds, by making different actions
