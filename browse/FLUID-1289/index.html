    Description

    When rolling over the tab the cursor changes from arrow to hand on some occasions, notices that the cursor really only changes rolling over from right to left.

    Steps to reproduce:

    1) Open the browser and navigate to the specified URL:
    http://build.fluidproject.org/fluid/sample-code/reorderer/jquery-tabs/jquery-tabs.html

    2) Roll over the tabs starting from left to right or up and down and the cursor does not change style.

    3) Cursor will only change style when rolling over the tab going from right to left.

    Environments

    Opera 9.5 (Mac OS 10.5, Win XP)
