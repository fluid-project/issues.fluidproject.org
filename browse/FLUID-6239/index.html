    Description

    Currently there are icon fonts used for the overview panel, uploader and prefs framework. These should all be transitioned to make use of the Infusion-Icons repo and grunt-webfont to construct the font at build time. This makes it easier to maintain and add new icons later, keep track of the PUA codes used in the font, keep the font as small as possible, and to easily reference what icons are included and what their source is.

    See: https://wiki.fluidproject.org/display/fluid/Standard+workflow+in+maintaining+and+creating+icon+fonts
