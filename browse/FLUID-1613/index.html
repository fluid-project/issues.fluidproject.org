    Description

    Undo button goes under the non-separated text when after you edit the editable text field.

    Steps to reproduce:

    1. Open URL: http://build.fluidproject.org/infusion/demos/inlineEdit/

    2. Using mouse, select one of inline edit text fields.

    3. Edit the text with a large number of characters with no gaps in between them.

    4. Select on blank/white space outside of editable text field
