    Description

    Steps to reproduce:

    1. Open the conditional adjusters - single panel example
    2. Open the prefs editor
    3. Using the keyboard, place focus on the "Magnification" switch and activate it
    4. Using the keyboard press the "tab" key.
    5. Notice that focus starts from the "Speak Text" switch instead of from the "Magnification" switch.

    Comments

    • Justin Obara commented 2021-07-06T08:09:45.042-0400

      This is likely due to the entire panel being re-rendered when the switch is activated; which causes the focus position to be lost.
