    • Giovanni Tirloni commented 2018-08-15T11:16:29.407-0400

      Created a pre-checkout hook script that cleans those files:

      #!/bin/bash -ex

# Safety check to ensure we're not being fed a bad path
if [[ ${BUILDKITE_BUILD_CHECKOUT_PATH} == /var/lib/buildkite-agent* ]]; then

  # Delete root-owned files that might have been created by an old containerized build
  docker run --rm -v ${BUILDKITE_BUILD_CHECKOUT_PATH}:/src busybox sh -c "cd /src && find /src/ -user root -exec rm -rf \{\} \;"

fi

      Successful build: https://buildkite.com/fluid-project/kettle/builds/61
