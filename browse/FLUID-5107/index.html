    Description

    The Sakai UIO integration demo was built to show UIO in the context of a real application. At this point, UIO has been integrated and deployed in many places and this demo no longer serves a purpose. In fact, we so rarely look at this demo that we often find that we've broken it inadvertently.

    We should remove this demo and point anyone who is interested in seeing UIO in context to:
    http://inclusivedesign.ca/
    http://www.oercommons.org/
    http://demo.floeproject.org/integration-demos/uiOptions/ClimateChange.html
