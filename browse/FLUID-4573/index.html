    Description

    The Renderer currently doesn't seem to handle a toggle button i.e. a <button> element that has a pressed state (i.e. "aria-pressed"). This is semantically different from a checkbox. For example, the "Play/Pause" control for the VideoPlayer is more properly implemented as a toggle button instead of a checkbox, but I'm unable to bind a <button> to the state in the model.

    Comments

    • Antranig Basman commented 2015-06-10T20:23:18.018-0400

      In the "new renderer" this will be handled by a Fluid component of the new type which is associated with a boolean model field. The "old renderer" itself will not be improved further and all the functionality it currently offers (e.g. binding to native HTML functions in DOM elements) will be unbundled for general use.
