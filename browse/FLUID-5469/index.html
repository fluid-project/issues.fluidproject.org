    Description

    FSS was deprecated in Infusion 1.5 and slated for removal in 2.0. It should be removed from the repository completely.

    • Justin Obara commented 2014-07-25T09:25:19.870-0400

      More discussion and proposals were made on the fluid-work mailing list:
      http://fluid.2324889.n4.nabble.com/Removing-FSS-from-Infusion-td9140.html

      The results are:

      1) Remove FSS
      2) Add Foundation to be used by demos, examples, and etc, but not by the components
      3) Add Normalize to be used by components, demos, and etc.
      4) See FLUID-5482 for using Stylus to generate contrast themes.
