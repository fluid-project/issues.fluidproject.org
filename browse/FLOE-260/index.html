    Description

    Integrate Text-to-Speech into the first discovery tool. This first pass should be able to be hooked up to read the contents of the tooltips as they are read. Additionally as a panel is displayed it should read the instruction text as well as indicating which panel is open and some additional help related text. The enabling/disabling of the text-to-speech should be hooked up to the a toggle button on the interface.

    See:
