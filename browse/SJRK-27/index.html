    • Gregor Moss commented 2019-01-02T18:35:40.210-0500

      Took the first half of the previous comment and opened SJRK-170 to deal with it.

      Regarding NVDA not reading the "alt text", which is to say the text that is meant to be displayed when no video file is able to load, that whole idea doesn't conform with the W3C's video accessibility suggestions anyway: https://www.w3.org/2008/06/video-notes

      A workable transcript and captions seem to be the best way at this point, so we should focus on that
