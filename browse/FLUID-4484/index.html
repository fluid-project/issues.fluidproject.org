    Description

    When using the keyboard to move focus between files in the file queue they do not scroll into view.

    Steps to recreate:

    1. Add several files to be uploaded.
    2. Shift tab to the file queue.
    3. Use the down arrow key to move focus to a file that is not currently visible.

    Notice that the file will get focus but does not scroll into view.

    Environments

    IE 8 XP, Win 7
    Fine in FF 6 Win 7, IE9 Win 7
