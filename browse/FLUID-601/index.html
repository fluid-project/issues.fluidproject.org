    Description

    dragging and dropping a thumbnail directly over a drop marker (the red vertical bar) does not work. Releasing on top of the drop marker will "snap" the dragged orderable back to its original location.

    This occurs if the mouse pointer is positioned directly over a drop marker.

    Comments

    • Michelle D'Souza commented 2008-05-16T09:57:04.000-0400

      In the lightbox, the container holding the thumbnails is not a drop target. Dropping on the drop marker is the same as dropping on the container which is invalid.

      The bug here is that the drop marker should disappear when the mouse is no longer hovering over a valid drop target.
