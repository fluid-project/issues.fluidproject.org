    Description

    This appears to be a Firefox specific issue.

    This was initially found in The Accessibility Exchange which uses the Orator component from Infusion for its self voicing. See issue #1633

    Steps to reproduce:

    1. Using Firefox open the Prefs Framework demo
    2. Open UIO and enable the text-to-speech preference
    3. Using the mouse make a selection of text that includes an SVG element
      1. For example the NOAA logo at the top left of the page
    4. Notice the error
