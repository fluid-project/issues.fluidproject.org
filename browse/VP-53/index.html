    Description

    Captionator is not notified when settings are modified in UIO. Showing captions are not restyled appropriately, however, the next caption to show after a setting is changed is show correctly.

    • Anastasia Cheetham commented 2013-05-09T15:00:14.481-0400

      Captions don't respond to changes in language. The selected language in the captions menu is successfully modified, but if the captions are visible at the time of the change, the actual caption text on the screen doesn't reflect the new language choice until the caption is hidden and then re-shown.

      To reproduce:
      1) Show captions and click the scrub bar so that the video is positioned somewhere that captions are on-screen.
      2) Use UIO to change the caption language

      • note that the highlighted language in the caption menu has been correctly updated
      • note that the onscreen caption text language has NOT been correctly updated
        3) Hide and then show captions
      • note that now, the captions are in the correct language

      Note that transcripts are fine, and do correctly respond to UIO changes in language preference.
