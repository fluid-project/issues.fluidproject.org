    Description

    The Read Me should mention that the Uploader does not currently work with the Flash 10 beta. This bugs is caused by Adobe changing the security protocols for Flash 10 which blocks the SWFUpload code from calling the OS File browser. The SWFUpload community is looking for fixes and work arounds both in their own code and in Flash, and we anticipate a work around or fix before Flash 10 goes final.

    Environments

    All browsers, all platforms, Flash 10 beta
