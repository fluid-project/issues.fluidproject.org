    Description

    If CTRL is depressed to move a tab and then ENTER is pressed (CTRL+ENTER) causes the tab example to temporarily break.

    Proper behaviour is restored by pressing TAB key and focus is placed on a orderable tab.

    Environments

    FF2 (Mac OS 10.5, Mac OS 10.6, Win XP)
