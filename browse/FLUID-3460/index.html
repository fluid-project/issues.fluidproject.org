    Description

    After the "Select all" button is activated (and all modules and dependencies are highlighted) you get auditory confirmation that all modules are selected, but in order for this to happen, you have hit Enter twice (as opposed to once).

    After "Reset selection" button is activated, it might be good to have confirmation that all modules are unchecked, i.e: "Modules Selected: none". My reasoning is because visually there is a change, so this should be communicated by JAWS too.

    Environments

    JAWS IE8
