    Description

    The new Uploader behavior to fix FLUID-822, means that the currently uploading file must complete or error before the Upload process can be stopped.

    There currently is no user feedback as to what is happening during the time that the user clicks the Stop Upload button and the upload actually stops. This makes the component appear broken and is potentially confusing.

    Please note that this would only effect large files and/or slow connections.
