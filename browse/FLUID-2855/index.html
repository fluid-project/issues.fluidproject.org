    Description

    Can't see the slider bar in "Medium Contrast Grey Scale" theme

    Steps to reproduce:

    1) Open the the sakai mock-up from the daily build site

    2) Change the theme to "Medium Contrast Grey Scale" and save the changes

    3) Re-open the UI Options Dialog

    Notice that you can see the slider thumb, but not the slider bar
