    Description

    Right now the front page graphic path is hard-coded into the front-page.php code and the image included with the theme.

    The wp-a11y theme should instead be configurable so user can easily upload their own graphic and change the front image.

    Comments

    • Jonathan Hung commented 2018-06-22T10:36:07.478-0400

      Currently there are two ways to do this:

      1. Upload the graphic using the built-in media uploader, find its path, then use the theme editor to change the path in front-page.php.
      2. Upload the image into the theme's image directory and replace the existing image. Or rename it and adjust the front-page.php file.
