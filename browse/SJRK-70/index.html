    Description

    Investigate and, if deemed appropriate, implement the selection of languages at a block level. If not block level, then revert to the previous behaviour of setting language at the story level.

    Consideration should be paid to:

    • the user experience of selecting language by block during editing,
    • the presentation of blocks of differing languages (RTL scripts, different character sets, etc.),
    • how each block will be read aloud
