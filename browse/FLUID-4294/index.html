    • Cindy Li commented 2011-06-17T16:20:13.507-0400

      At page refresh, the UI Options settings "emphasize links" and "make input larger" are applies to the whole page except UI Options interface itself. This is because fluid.uiOptions is instantiated and renders its interface after the instantiation of fluid.uiEnhancer. At the time that these 2 settings look for <a> or <input> tags inside <body>, the ones inside UI Options itself are not rendered yet. Other settings are applied successfully because those css classes are applied to uiEnhancer container, which is <body>.
