    Description

    The Dockerfile currently specifies Node 10.22.1, which will be moving out of LTS in April 2021. As Node 14 will be entering LTS in late October 2020, it makes sense to update to 14 rather than 12, which is the current LTS at time of filing.

    This task involves ensuring the site runs properly in a Node 14 development environment as well as with a Node 14 based Docker image & server environment.
