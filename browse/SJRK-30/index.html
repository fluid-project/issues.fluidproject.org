    • Alan Harnum commented 2017-11-09T16:01:39.140-0500

      Consider ways to make the story authoring via images only, if the author decided to do it that way.

      Perhaps choosing from a set of cards or symbolic images? Communication boards. How would we decide what is in the set of images? Adding images is not the same as choosing from a predefined set of pictograms

      Accessibility considerations relating to image use:

      • alternative text input for each image
      • record audio which is transcribed to provide alternative text
      • we should encourage authors to provide alternative content, keeping in mind the complexity of the interface
