      When updating jQuery UI we should take care to look at re-implementing the tooltip and tabs wrapper components. These should be brought up to modern standards. In particular for tooltip we should avoid the current idiom of creating new tooltips for each instance, rather we should share a single tooltip and move it around. Additionally, we should make sure that it supports all the modern features, e.g. custom-content (http://jqueryui.com/tooltip/#custom-content)
