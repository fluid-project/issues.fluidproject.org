    Description

    The uPortal integration demo was originally created to show several components on a single page. That need has evaporated as Infusion has been used more widely and the Fluid community has moved to creating larger applications.

    We also used the demo to test some features that we wanted to ensure would continue to work such as nested Reorderers. When we remove the demo, we should create tests to cover these features.
