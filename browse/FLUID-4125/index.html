    Description

    HTML5 currently doesn't support MIME type extensions as acceptable values in the accept attribute. MIME type extensions are fully ignored at the moment. Only MIME types are valid. There is no word on whether or not MIME type extensions will be supported as they are conflicting across operating systems.

    See the following for more details about the issue:

    https://bugzilla.mozilla.org/show_bug.cgi?id=565274

    https://bugzilla.mozilla.org/show_bug.cgi?id=377624
