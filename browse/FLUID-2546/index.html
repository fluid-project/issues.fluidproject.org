    Test 2: Many Characters (Not Separated)

    • Procedure
      1. Open the browser and navigate to the specified URL
      2. Using the mouse, select click one of the inline editable texts
      3. Edit the text with a large number (e.g. 1000 characters) of characters without any gaps between
      4. Using the mouse, click on a blank spot outside of the editable text field
    • Expected Results
      o The inline editable text should be changed to the new value
      o An 'undo edit' link should appear next to the inline editable text
    • Stop Test
      o Refresh the browser to return the page to its initial state

    I was unable to find the Undo edit link.
