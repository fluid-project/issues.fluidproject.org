    Description

    One of the original requirements for the GPII ProcessReporter was that it support detection of a solution changing state from running to not-running and vice versa. At present, this functionality is no longer desired by GPII. But, it has been suggested that said functionality should be moved into infusion for more general usage, i.e. not just processes, but anything that changes state.
