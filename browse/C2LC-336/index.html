    Description

    Below are some examples of the current behaviour.

    Hide Backward 3

    Current:

    B1   F2

    B2   F3

    L45  R45

    L90  R90

    L180 R180

    F1

    What we want:

    B1   F1

    B2   F2

    *    F3

    L45  R45

    L90  R90

    L180 R180

    Hide Backward 2 and Backward 3

    Current:

    B1   F3

    L45  R45

    L90  R90

    L180 R180

    F1

    F2

    What we want:

    B1   F1

    *    F2

    *    F3

    L45  R45

    L90  R90

    L180 R180

    Hide all Backwards commands

    *    F1

    *    F2

    *    F3

    L45  R45

    L90  R90

    L180 R180

    Hide all except for Forward 1, Backward 1, Left 90, Right 90

    Current:

    B1

    L90

    F1

    R90

    What we want:

    B1  F1

    L90 R90

    or:

    B1

    F1

    L90

    R90

    Tabbing order

    We would like to tab through the blocks in the following order:

    • F1, F2, F3
    • B1, B2, B3
    • L45, L90, L180
    • R45, R90, R180

    But keep the current visual order.
