    Description

    In the past, we did a proof of concept implementation of metadata annotations inside an authoring and viewing system.

    http://wiki.fluidproject.org/display/fluid/(Floe)+OER+Commons+Access+for+All+metadata+design

    We now need to generalize this work and build it in a way that it can be integrated into production environments.

    Designs for this work have been started here:

    http://wiki.fluidproject.org/download/attachments/37854510/FLOE-metadata-design-1a-13.png

    We need to implement a panel that displays availability or unavailability of metadata that is shown as "Your video features" panel at the top right corner of the design.
